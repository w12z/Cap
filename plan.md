# Cap — 轻量经济策略游戏开发计划

一个类似 Capitalism Lab 的轻量经济策略游戏。视觉风格参考「跳一跳」：低多边形、
正交/等距视角、柔和马卡龙配色、软阴影、极简 UI。使用 **Godot 4（GDScript）**。

## 核心设计目标

- 数个虚构国家开局，各有不同经济体制与初始条件（资本主义发达国、社会主义国等）。
- 国家经济影响公司；公司后期反向影响国家经济（**双向**）。
- 多种玩家出身（富二代 / 白手起家等）→ 选择初创公司方向。
- **严谨的经济系统优先**：央行、商业银行、保险、证券市场、供需出清市场。
- 初期公司管理功能从简，经济环境/市场建构先完善。

渲染：Forward+；正交相机；DirectionalLight 软阴影；WorldEnvironment + 程序化天空渐变；
SSAO 软接触阴影；微弱 Glow。

## 技术架构（纯 2D 等距）

```
scenes/Main.tscn            根场景（Node2D + Main.gd）
scripts/Palette.gd          全局配色 + 等距瓦片几何常量（class_name Palette）
scripts/Main.gd             渐变背景、Camera2D、WorldGrid、HUD
scripts/WorldGrid.gd        网格容器：等距坐标映射、生成瓦片、选中管理（避开内置 GridMap 类名）
scripts/Tile.gd             单格：Area2D，_draw 画 顶面菱形+两侧面(明暗)+软阴影，z_index=gx+gy 遮挡排序
```

每个瓦片以 2:1 菱形绘制：顶面为基色，右侧面 ×0.80、左侧面 ×0.60 模拟受光；
地面绘制半透明椭圆软阴影；悬停/选中时整体上抬（lift）并切换配色，靠 `z_index` 实现等距遮挡。
